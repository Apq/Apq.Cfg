添加环境变量配置源

public CfgBuilder AddEnvironmentVariables(int level, string? prefix = null)

Parameters

level int

配置层级，数值越大优先级越高

prefix string

环境变量前缀，为null时加载所有环境变量

Returns

CfgBuilder

配置构建器实例，支持链式调用

Examples

var cfg = new CfgBuilder()
    .AddJson("config.json", level: 0)
    .AddEnvironmentVariables(level: 1, prefix: "APP_")
    .Build();
