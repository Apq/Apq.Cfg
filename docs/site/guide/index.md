# Apq.Cfg 简介

Apq.Cfg 是一个强大、灵活的 .NET 配置管理框架，提供统一的 API 来管理来自多种来源的配置数据。

## 特性概览

### 🔌 多配置源支持

支持丰富的配置源类型：

**本地配置源：**
- JSON - 最常用的配置格式
- YAML - 人类友好的配置格式
- TOML - 简洁明了的配置格式
- XML - 传统的配置格式
- INI - 简单的键值对格式
- ENV - 环境变量文件

**远程配置源：**
- Redis - 高性能键值存储
- Database - 关系型数据库
- Etcd - 分布式键值存储
- Consul - 服务发现和配置
- Nacos - 阿里巴巴配置中心
- Apollo - 携程配置中心
- Vault - HashiCorp 密钥管理
- Zookeeper - 分布式协调服务

### ⚡ 高性能

- 基于源代码生成器的零反射绑定
- 内置多级缓存机制
- 延迟加载和按需解析
- 经过基准测试验证

### 🔄 动态重载

- 支持配置热更新
- 文件变更自动检测
- 远程配置实时同步
- 变更通知和回调机制

### 🛡️ 类型安全

- 强类型配置绑定
- 编译时类型检查
- IDE 智能提示支持
- 自动类型转换

## 设计理念

1. **统一抽象** - 无论配置来自哪里，使用相同的 API
2. **可扩展性** - 轻松添加自定义配置源
3. **最小依赖** - 按需引用，避免不必要的依赖
4. **生产就绪** - 经过充分测试，适用于生产环境
