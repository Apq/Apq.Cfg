# Apq.Cfg.Crypto.DataProtection v1.0.6

基于 ASP.NET Core Data Protection 的加密提供者。

## 功能

- 使用 ASP.NET Core Data Protection API 进行加密
- 适用于 ASP.NET Core 应用程序
- 支持密钥轮换

## 使用示例

```csharp
var cfg = new CfgBuilder()
    .AddJson("config.json", level: 0)
    .AddDataProtectionEncryption(services)
    .Build();
```
