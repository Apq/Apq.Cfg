# Apq.Cfg.Env v1.0.0

发布日期：2025-12-27

## 新功能

- 初始版本
- 支持 .env 文件格式配置源
- 支持注释（以 `#` 开头）
- 支持双引号包裹的值（支持 `\n`, `\r`, `\t`, `\"`, `\\` 转义）
- 支持单引号包裹的值（原样保留，不处理转义）
- 支持 `export` 前缀
- 支持嵌套配置（使用 `__` 转换为 `:`）
- 支持文件变更自动重载
- 支持可写配置源
- 支持以点开头的文件名（如 `.env`）

## 使用示例

```csharp
using Apq.Cfg;
using Apq.Cfg.Env;

var cfg = new CfgBuilder()
    .AddEnv(".env", level: 0, writeable: true)
    .Build();

// 读取配置
var appName = cfg.Get("APP_NAME");
var dbHost = cfg.Get("DATABASE:HOST");  // DATABASE__HOST 自动转换
```

## .env 文件示例

```env
# 应用配置
APP_NAME=MyApp
APP_DEBUG=true

# 数据库配置（使用 __ 表示嵌套）
DATABASE__HOST=localhost
DATABASE__PORT=5432

# 支持引号包裹的值
MESSAGE="Hello, World!"
MULTILINE="Line1\nLine2"

# 支持 export 前缀
export API_KEY=secret123
```
