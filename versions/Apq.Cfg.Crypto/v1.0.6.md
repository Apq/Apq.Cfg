# Apq.Cfg.Crypto v1.0.6

配置加密脱敏扩展包。

## 功能

- 配置值加密/解密（支持 AES-GCM、AES-CBC、ChaCha20、RSA、SM4 等算法）
- 敏感配置脱敏（日志输出时自动隐藏敏感信息）
- 基于 BouncyCastle.Cryptography 实现

## 使用示例

```csharp
var cfg = new CfgBuilder()
    .AddJson("config.json", level: 0)
    .AddAesGcmEncryptionFromEnv()  // 从环境变量读取密钥
    .AddSensitiveMasking()          // 添加脱敏支持
    .Build();

// 读取时自动解密
var password = cfg.Get("Database:Password");

// 日志输出时脱敏
Console.WriteLine(cfg.GetMasked("Database:Password"));
// 输出: myS***ord
```
